## 华中科技大学校内电动车通行证（仿制版）
Bolaxious因为新车的牌照还没办无法办理校内通行证，担心放假时电动车没有通行证而被逐出校园，故而对着小红书的一篇博文的照片仿造了这个电动车通行证。
